# 株価と出来高の関係分析

## 概要
本プロジェクトでは、株価（終値）と出来高の関係を分析し、両者の関係性を明らかにすることを目標としている。

## 使用データ
- セブン&アイホールディングスの株価、出来高データ（Yahoo!ファイナンス）

## 分析内容
### 1. 可視化
- 株価（終値）と出来高を同一グラフ上に表示
- 時系列での動きを比較

### 2. 変化率
- 株価リターン（pct_change）を算出
- 出来高変化率を算出
- 相関係数を計算

### 3. 単回帰分析
- 出来高変化率 → 株価リターンの単回帰分析
- 決定係数（R²）を算出

### 4. ラグ分析
- 出来高の1日遅れ（lag）を導入
- 出来高の変化が翌日の株価に与える影響を検証

## 実行方法
- 必要なライブラリーをインストール

pip install pandas matplotlib scikit-learn openpyxl
- データファイル（stock-seveneleven.xlsx）
- コードを実行

## 分析結果
- 出来高変化率と株価リターン及び、出来高と株価の両者の相関係数は非常に小さく、両者の関係は見られなかった
- 単回帰分析においても決定係数（R^2）はほぼ0に近く、出来高から株価の変動を説明することは難しい
- ラグ（出来高を1日遅らせる）を導入した場合においても、明確な関係性は確認できなかった

以上より、出来高の変化では株価の変動を予測することが困難であることが示唆された。
しかし、株価は時事情報・業績・トレンドなど多くの要因に影響されるため、出来高のみでの分析では説明力が低いと考えられる。
